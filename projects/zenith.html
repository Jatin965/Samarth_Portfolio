The Zenith Project

The Zenith project is based on the fundamentals of Al, machine learning and computer vision. As the lead developer of Project Zenith I took a dive into many of these fields that are considered challenging and complicated to see for myself the maximum potential that I can achieve with nothing more than a rudimentary knowledge of Python.

This project consists of a voice assistant that is not only capable of handling mundane tasks but is also equipped with features like Face Recognition, Mood Recognition, Coding Guide etc. The collection of many of these unique features within a voice assistant, makes our project stand out among many others in the same category Built with Python
